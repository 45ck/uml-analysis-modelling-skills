# Sample Flow

## Prompt
Model a simplified online shopping system where customers browse products, place orders, pay, and track delivery.

## Suggested skill flow
1. scenario-to-uml-transformer
2. use-case-modeler
3. use-case-description-writer
4. domain-class-modeler
5. multiplicity-checker
6. activity-diagram-builder
7. sequence-diagram-builder
8. boundary-control-entity-mapper
9. design-class-modeler
10. model-consistency-checker

## Expected artifacts
- Use-case list and system boundary
- One or more structured use-case descriptions
- Domain class model notes
- Multiplicity review notes
- Activity diagram notes for checkout or order tracking
- Sequence diagram notes for place-order flow
- BCE mapping
- Design class model notes
- Consistency findings and open questions
